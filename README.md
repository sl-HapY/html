#  گیت‌گت دانلودر در شرایط تخمی برای گیتهاب اکشنز

یک اسکریپت تخمی که توی گیتهاب اکشنز کار می‌کنه و با نوشتن چند خط دستور، هرچی لازم داری رو دانلود می‌کنه و آخرش یه فایل ZIP تخمی تر میده بهت، مخصوص وقتی که شرایت تخمی تر تر هست و دسترسی به خیلی از سایت‌ها بسته‌ست.

---

## 🔧 دستورات قابل قبول

| دستور   | کاربرد                         | فرمت کلی                                                | مثال |
|---------|--------------------------------|--------------------------------------------------------|------|
| `wget`  | دانلود مستقیم فایل             | `wget <لینک> [o=نام‌خروجی]`                            | `wget https://example.com/file.zip o=myfile.zip` |
| `git`   | کلون مخزن Git (مثل مدل‌های HF) | `git <آدرس> [b=شاخه] [d=پوشه] [lfs=true] [depth=1]`     | `git https://huggingface.co/bert-base-uncased b=main d=bert lfs=true` |
| `mirror`| آینه‌سازی کامل سایت           | `mirror <آدرس> [o=پوشه] [depth=1000]`                  | `mirror https://example-docs.com o=docmirror depth=500` |

**توضیح پارامترهای اختیاری:**
- برای `wget`:  
  `o=...` : مسیر و نام خروجی (نسبت به پوشهٔ خروجی).  
- برای `git`:  
  `b=...` : نام شاخه (پیش‌فرض: شاخهٔ اصلی مخزن).  
  `d=...` : پوشه‌ای که مخزن درون آن کلون شود.  
  `lfs=true/false` : آیا فایل‌های LFS گرفته شوند؟ (پیش‌فرض `true`).  
  `depth=...` : عمق تاریخچه (پیش‌فرض ۱).  
- برای `mirror`:  
  `o=...` : نام پوشه‌ای که سایت در آن ذخیره شود.  
  `depth=...` : عمق خزیدن (پیش‌فرض ۱۰۰۰).

---

## روش استفاده

1. وارد مخزن گیتهاب خود شو و دو فایل را همان‌طور که هست بگذار:  
   - `downloader.py` در ریشهٔ مخزن  
   - `.github/workflows/download.yml` (همان workflow)
2. به تب **Actions** برو، **Multi Downloader** را انتخاب کن.
3. روی دکمهٔ **Run workflow** کلیک کن.
4. در کادر بزرگ، دستورات خود را خط‌به‌خط بنویس (هر خط یک دستور).
5. (اختیاری) نام فایل ZIP خروجی را در فیلد دوم تغییر بده.
6. دکمهٔ سبز **Run workflow** را بزن.
7. بعد از پایان اجرا (علامت ✔ سبز)، به بخش **Artifacts** برو و فایل ZIP را دانلود کن.

> **نکته:**  
> در فایل YAML نیز می‌توانی مقادیر پیش‌فرض دستورات را مستقیماً عوض کنی تا هر بار نیاز به تایپ نباشد.

## 📌 نمونه‌های ترکیبی

**دانلود یک فایل مستقیم + کلون یک مدل:**
```
wget https://example.com/data.zip o=mydata.zip
git https://huggingface.co/google/vit-base-patch16-224 b=main d=vit lfs=true depth=1
```

**میرور کردن چند سایت همزمان:**
```
mirror https://docs.python.org/3/ o=python_docs depth=2000
mirror https://example-site.com o=site2
```
